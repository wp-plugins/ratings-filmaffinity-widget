Ratings FilmAffinity Widget shows the latest movies as voted by you in FilmAffinity.



== Description ==

= English =
Ratings FilmAffinity Widget shows the cover and information about the latest movies as voted by you on page FilmAffinity.

You can choose from various display options such as:

*   FilmAffinity language in English or Spanish.
*   Grid visualization in: 1x1, 1x2, 1x3, 2x1, 2x2 y 2x3 (rows x columns)
*   Update time films.
*   Show user.
*   Show the average rating.
*   Show the total votes.
*   Show note movies.
*   Show the description of the film.

= Spanish =
Ratings FilmAffinity Widget le permite mostrar un widget en la barra lateral de su página con las últimas películas votadas por usted en la web de FilmAffinity.

Puede elegir entre varias opciones de visualización como:

*   Idioma de FilmAffinity en Español o Ingles.
*   Rejilla de visualización de: 1x1, 1x2, 1x3, 2x1, 2x2 y 2x3 (filas x columnas)
*   Tiempo de actualización de las películas.
*   Mostrar el usuario.
*   Mostrar la media de votos.
*   Mostrar el total de votos.
*   Mostrar la nota de las películas.
*   Mostrar la descripción de las películas.



== Installation ==

= Install =
1. Upload the 'Ratings FilmAffinity Widget' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Widgets -> Ratings FilmAffinity Widget and move to sidebar.

= Uninstall =
1. Deactivate Ratings FilmAffinity Widget in the 'Plugins' menu in Wordpress.
2. After Deactivation a 'Delete' link appears below the plugin name, follow the link and confim with 'Yes, Delete these files'.
3. This will delete all the plugin files from the server as well as erasing all options the plugin has stored in the database.



== Frequently Asked Questions ==

= English =
= Does the widget you can refresh more often? =
No. With each update there that get data from FilmAfinity, if you refresh too often could be a problem.

= Spanish =
= ¿Se pueden actualizar las peliculas del widget con más frecuencia? =
No. Con cada actualizacion hay que obtener todos los datos de FilmAffinity, si se actualizase muy a menudo podria haber problemas.
